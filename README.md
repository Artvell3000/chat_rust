# Консольный чат

Консольное приложение на Rust, которое работает в двух режимах: сервер и клиент.
## Запуск

### Сервер

```sh
cargo run -- server --bind 0.0.0.0:9000
```

Логи сервера всегда включены.

### Клиент

```sh
cargo run -- client --addr 127.0.0.1:9000 --name artem
```

## Формат сообщений

Клиент отправляет сообщения в формате:

```
<username>: <message>
```

Сообщения передаются построчно и разделяются символом перевода строки.

## Подсветка упоминаний

Если входящее сообщение содержит подстроку `@<USERNAME>`, строка подсвечивается в консоли.

## Обработка ошибок

- Клиент: при невозможности подключения выводит понятную ошибку и завершает работу с ненулевым кодом.
- Клиент: при разрыве соединения или ошибке чтения/записи сообщает об этом и завершает работу.
- Сервер: ошибка у одного клиента не приводит к остановке сервера; проблемное соединение закрывается.
